# 🌡️ Temperature Converter

A simple web-based Temperature Converter that converts **Celsius (°C)** to **Fahrenheit (°F)** and **Kelvin (K)** using standard temperature conversion formulas.  
This project is designed with a clean and user-friendly interface and is suitable for beginners learning web development.

---

## 🚀 Features

- Convert **Celsius to Fahrenheit**
- Convert **Celsius to Kelvin**
- Simple and responsive user interface
- Instant result calculation

## ⚙️ How It Works

The converter uses the following formulas:

**Celsius → Fahrenheit**

```
F = (C × 9/5) + 32
```

**Celsius → Kelvin**

```
K = C + 273.15
```
